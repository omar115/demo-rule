Installation¶

You can install sphinx-multiversion via `pip <pip_>`_ <pip_> or directly from setup.py.

Install from PyPi¶

Using pip you can easily install the latest release version `PyPI <pypi_>`_ <pypi_>:

pip install sphinx-multiversion

Install from GitHub¶

You can install the latest development version using pip directly from the GitHub repository:

pip install git+https://github.com/Holzhaus/sphinx-multiversion.git

Install via setup.py¶

It’s also possible to clone the Git repository and install the extension using its setup.py:

git clone https://github.com/Holzhaus/sphinx-multiversion.git
cd sphinx-multiversion
